## Sobre o Projeto
O TerraSense é um aplicativo mobile desenvolvido para auxiliar no monitoramento da vegetação em rodovias.

O sistema permite acompanhar trechos críticos, visualizar sensores, registrar ocorrências e auxiliar equipes de manutenção.

---

# Problema

Vegetação alta nas rodovias pode causar:

- Redução da visibilidade
- Dificuldade na identificação de placas
- Riscos de acidentes
- Falta de monitoramento eficiente

---

# Solução

O TerraSense permite:

- Monitoramento de vegetação
- Registro de ocorrências
- Classificação de risco
- Controle de manutenção
- Visualização de dados em tempo real
